# Context: Web Dashboard

## Domain

Streamlit 可视化面板，展示持仓分析结果、风险指标、操作推荐。

## Key Concepts

- **持仓概览（Portfolio Overview）**：持仓分布、收益/亏损、风险暴露
- **分析报告（Analysis Report）**：AI 生成的详细分析报告
- **操作建议（Trade Recommendations）**：买入/卖出/持有/加仓/减仓建议
- **风险仪表盘（Risk Dashboard）**：波动率、最大回撤、相关性、VaR
- **历史追踪（History Tracking）**：决策记录、绩效回溯
