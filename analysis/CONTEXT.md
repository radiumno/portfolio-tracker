# Context: Analysis Engine

## Domain

基金/ETF 智能分析核心引擎。7 阶段分析流水线：数据采集 → 标的体检 → 组合分析 → 理论评估 → 辩论 → 风控 → 推荐。

## Key Concepts

- **ETF 分析器**：跟踪误差、费率侵蚀、流动性、折溢价、行业暴露
- **基金分析器**：经理评价、Brinson 归因、风格漂移检测、同类排名
- **风险分析器**：波动率、最大回撤、VaR/CVaR、边际风险贡献
- **投资理论**：价值投资、成长投资、全天候、量化因子、行为金融（可插拔 BaseTheory）
- **辩论引擎**：3 阶段辩论（持仓结构/调仓审视/优先级排序）
- **决策引擎**：综合多源分析输出操作推荐

## Pipeline

P1 数据采集 → P2 标的体检 → P3 组合分析 → P4 理论评估 → P5 辩论 → P6 风控 → P7 推荐
